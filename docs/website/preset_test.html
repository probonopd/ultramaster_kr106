Preset A/B Test MIDI Files

Help us calibrate the KR-106

We're building the most accurate Juno-106 emulation possible, and we need recordings from real hardware to get there. These MIDI files program your 106 to each factory preset via SysEx and play a sequence of test notes. You don't need the original patches in memory -- the files set every parameter directly.

If you have a Juno-106, please record a few banks and send us the audio. Even a handful of presets helps. The more units we hear from, the better we can model the component variance between machines.

Submit recordings via GitHub

How to record

  1. Connect your Juno-106 MIDI in (USB interface or DIN cable)
  2. Download a bank file below (8 presets each, fire and forget)
  3. Play the MIDI file into the 106 and record the audio output
  4. Upload the recording to a GitHub issue, or email it to us

Record at 96 kHz if you can, 44.1 kHz is fine too. Line out is best (no amp/speaker coloring). Include your unit's serial number if you know it -- helps us track variance between machines.

Restore your factory presets

Lost your factory patches? These files also work as a preset loader. Play any file into your 106 and it will set every slider and switch to the factory values. The individual preset files below let you audition any of the 128 patches.

What each file plays

  • SysEx to set all parameters (no Program Change, won't overwrite your patches)
  • One long C4 note (hold time matches the preset's attack + 500ms sustain)
  • Full release tail (matches the preset's release time)
  • Five quick notes across the keyboard: C2, C3, C4, C5, C6

VCF Calibration Test

This file sweeps white noise through the filter at 9 cutoff frequencies (C1-C9) and 11 resonance levels (0.0-1.0), 1 second per step. 99 test points, about 100 seconds total. Noise-only patch, gate mode VCA, no envelope modulation. Ideal for measuring the filter's frequency response and resonance behavior.

Download filter_test.mid

Technical Notes

  • Standard Roland Juno-106 SysEx (F0 41 32 ...). Any 106 should respond.
  • No Program Change is sent -- your stored patches are not affected.
  • Note timing adapts to each preset's attack and release envelope settings.
  • Works with Juno-106 only (not Juno-60, which uses a different SysEx format).
